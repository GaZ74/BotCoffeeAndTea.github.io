Project: My Favorite Coffee Shop Website

Author: Đặng Hoàng Minh Long
Date: 19/10/2025

Description:
=============================================================================
this project is a simple 6-page website for a local coffee shop. It includes:
* Home page (index.html) introducing the shop
* About page describing the business
* Contact page with an order and feedback form 
=============================================================================

Files Included:
============================================
* index.html - main homepage
* about.html - business background and story
* Contact.html - Customer's order feedback
* Gallery.html - some view of the coffee shop
* Map.html - Location of the coffee shop
* menu.html - Menu of the coffee shop
* style.css - stylesheet used for all pages
* images/ - folder containing website images
============================================

How to view the website:
================================================================
1/ Open index.html in a web browser (e.g., Chrome, Edge, Opera).
2/ Navigate through pages using the navigation bar links.
================================================================

Tools used:
* HTML5
* CSS3
* Visual Studio Code (for editing)
